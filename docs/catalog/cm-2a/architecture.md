# CM-2A Catalog Foundation

CornerOps produces a checksum-pinned export package. CornerMex validates and classifies every source row, previews without writes, and imports only with an exact fingerprint, reviewed commit, target project and idempotency key. Imported products remain `draft`, variants inactive, stock zero and invisible to public policies. Blocked rows remain visible only to founder/admin through `catalog_import_reviews`.
